# Growth Marketing Research Repository

## Overview

This repository contains research collected from leading practitioners in Growth Marketing, SEO, AEO (Answer Engine Optimization), GEO (Generative Engine Optimization), Content Marketing, and AI-driven organic growth.

The goal of this project is to understand how modern marketers are adapting to AI search, content distribution, audience acquisition, and evolving search behaviors.

This research serves as the foundation for building a practical growth marketing playbook based on insights from experienced practitioners and industry leaders.

The focus is on identifying repeatable growth frameworks, content strategies, and distribution methods that can be applied in modern marketing environments.
